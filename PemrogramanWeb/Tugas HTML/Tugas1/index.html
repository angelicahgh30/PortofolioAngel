Mengenal HTML, CSS, JavaScript & Tailwind

HTML

HTML (HyperText Markup Language) adalah bahasa utama yang digunakan untuk membuat struktur halaman web. Semua teks, gambar, dan elemen di internet disusun menggunakan HTML sebagai kerangkanya.

CSS

CSS (Cascading Style Sheets) berfungsi untuk mempercantik tampilan halaman web. Dengan CSS, kita bisa menata warna, ukuran, tata letak, dan efek visual agar tampak menarik dan responsif di berbagai perangkat.

JavaScript

JavaScript memberikan interaktivitas pada website. Dengan bahasa ini, kita bisa membuat animasi, tombol yang bereaksi, hingga fitur dinamis seperti form dan kalkulator yang berjalan langsung di halaman.

Tailwind CSS

Tailwind CSS adalah framework modern untuk mempercepat proses styling. Cukup gunakan kelas siap pakai tanpa harus menulis CSS panjang, hasilnya tetap rapi dan profesional.
